项目说明（简版）：

技术栈：Vite + Vue3 + TS，已全局引入 Element Plus 样式；静态首屏模板放在 public/html5up-aerial。
入口逻辑：首次访问显示 Aerial 首屏（public/html5up-aerial/index.html，淡入，点击“点击进入”淡出后进入主站）；进入主站后为单页组件切换，底部固定页脚。
项目结构（主要目录/文件）：

src/main.ts：挂载入口，引入全局样式与 Element Plus CSS。
src/App.vue：导航 + 当前内容组件切换 + 底部页脚；初始显示 AerialLanding，收到 enter-main 消息后切换主站。
src/components/
AerialLanding.vue：iframe 加载首屏模板。
Navbar.vue：顶部导航（切换 home/about/services/activities/recruitment/network）。
BannerSection.vue：首页 Banner。
ServicesSection.vue：服务中心。
AboutSection.vue：兴趣小组.
ActivitiesSection.vue：活动动态（卡片与筛选）。
RecruitmentSection.vue：招新表单。
NetworkSection.vue：校园网服务与反馈。
FooterSection.vue：页脚信息。
src/style.css：全局基础样式。
public/html5up-aerial：首屏静态模板及资源（CSS/字体/图片）；index.html 中“点击进入”按钮触发淡出并向父页 postMessage 进入主站。
public/logo.png 等：站点 logo；其余资源按占位图使用，可替换为真实图片。
页面对应中文：

首屏：Aerial 模板（“点击进入”按钮）。
首页 Banner：技术服务校园介绍。
服务中心：内修/外修/星印云。
兴趣小组：组织简介与分组.

活动动态：活动卡片与分类按钮。
招新专栏：招新说明与报名表单。
校园网服务：联系方式与问题反馈表单。
页脚：导航与相关链接.
